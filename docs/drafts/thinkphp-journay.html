About urls and directories In example, in the directory <app>/application, the structures look like this. The "demo" and "index" directories would be the modules (模块). In case of "demo", one page or url will be one method of Index class in demo/controller/Index.php. 

<?php
namespace app\demo\controller;

class Index
{
    public function index()
    {
    // http://localhost:8000/demo
    // or
    // http://localhost:8000/demo/index
    // or
    // http://localhost:8000/demon/index/index
